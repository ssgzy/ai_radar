# Prompt 设计

关联：[[Agent 设计]] | [[模块说明]] | [[问题与坑点]]

## 当前目标

- 输出中文
- 通俗易懂
- 不只是压缩内容，还要说明为什么值得关注
- 对 `arXiv` 允许处理完整摘要
- 不要生成过多 Markdown 装饰符号

## 当前结构

- `system_prompt`：限定整体风格和输出要求
- `summary_template`：强制固定标题
- 当前固定标题包括：
  - 内容概述
  - 解决的问题
  - 为什么值得关注
  - 适合我关注的原因
  - 关键词
  - 简洁引用

## 当前取舍

- 不强制 JSON 输出，避免本地模型格式跑偏时完全失效
- 改为固定中文标题，解析更稳
- 在解析阶段补充清洗逻辑，去掉 `**` 等噪音
