## 项目目标

这个项目要构建一个本地运行的 AI Radar / AI Research Intelligence System。目标不是只抓 arXiv，而是逐步形成一个可持续迭代的个人 AI 情报系统，覆盖采集、处理、评分、去重、日报生成和 Obsidian 沉淀。

## 当前范围

当前已完成 V2 结构化增强版本：

- 初始化项目目录
- 初始化 Obsidian 中文笔记体系
- 跑通 arXiv collector
- 跑通 GitHub collector
- 跑通本地 Ollama summarizer
- 输出 processed JSON
- 输出 scored JSON
- 输出 Obsidian Markdown 和排序日报
- 增加 scoring / tags / 推荐级别
- 保证运行中有清晰进度和阶段日志
