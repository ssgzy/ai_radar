# 数据说明

关联：[[系统架构]] | [[结果索引]] | [[模块说明]] | [[Exporter 设计]]

## 数据分层

- `data/raw/`：采集器原始输出，按来源分目录
- `data/processed/`：模型处理后的结构化结果
- `data/processed/scored/`：按总分排序的结构化结果
- `data/state/`：运行状态、已见集合、来源检查点
- `outputs/obsidian/`：面向阅读的 Obsidian 笔记和日报
- `outputs/debug/`：prompt、response 和后续失败条目

## 当前已落盘的数据

- `arXiv` raw JSON
- `GitHub` raw JSON
- `arXiv` processed JSON
- `GitHub` processed JSON
- 合并后的 merged JSON
- 按分数排序的 scored JSON
- `last_run.json`
- `seen_ids.json`
- `seen_urls.json`
- `seen_titles.json`
- `source_checkpoints.json`

## 使用原则

- 原始数据尽量不修改，方便回溯
- 处理后数据允许后续重新生成
- 人类阅读优先看 `outputs/obsidian/`
- 机器复用优先看 `data/processed/`
